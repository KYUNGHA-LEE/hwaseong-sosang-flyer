# 소상공인 AI 교육 5주차 핸드아웃 (hwaseong-sosang-flyer)

"소상공인을 위한 AI 매출 도구 만들기" 교육 5주차 수업에서 나눠 주는 웹 핸드아웃입니다. 종이 유인물 대신 링크 하나로 수강생에게 전달할 수 있습니다.

## 담긴 내용

- 오늘 들고 갈 매출 도구 세트 정리
- 오늘 쓸 도구 2종과 무료 사용 한도 안내
- 업종별 헤드라인 연습 (내가 먼저 쓰고 AI는 코치)
- 과장광고 5종 금지 점검표
- 90분 수업 설계 가이드 (가르치는 사람 되기)
- 복사해서 바로 쓰는 수업 설계 프롬프트
- 카피 10줄 노트
- 5주 과정 마무리 OX 퀴즈
